pwaOS Documentation

pwaOS is an Progressive Web App standard-based runtime library, it is free! The pwaos runtime is implemented as Service-Oriented Architecture, Its client API support C/C++, javascript, Rust, java, mojo, c#.pwaOS runtime is managed by pwaOS team, but is available for use by anyone under the BSD Free Software License. While engineering of the core components is done by the pwaOS development team, we consider contributions from any source.

The site you are at is the API documentation for the pwaOS runtime client library. Other resources are:

  • Download demo app: download.pwaos.net.
  • Example and develop static library: guide.pwaos.net.
  • Offical website: pwaos.net.
  • Contact: pwaOS team.
